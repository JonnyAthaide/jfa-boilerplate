# Coding Standards

## JavaScript

- ES Modules
- Barrel Exports
- Uma classe por responsabilidade

## Estrutura

Toda pasta deve possuir index.js.

Sempre importar pelo barrel.

Nunca importar arquivos internos de outro módulo.

## Nomeação

PascalCase

Classes

camelCase

Funções

UPPER_CASE

Constantes

kebab-case

Arquivos

## Sass

Utilizar tokens.

Evitar sobrescritas diretas.

Sempre reutilizar componentes.

## Comentários

Comentar apenas decisões importantes.

Não comentar código óbvio.
