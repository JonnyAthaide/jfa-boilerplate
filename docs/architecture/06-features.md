# Features

## Objetivo

Implementar funcionalidades reutilizáveis compostas por múltiplos módulos do framework.

---

## Responsabilidades

Orquestrar Components.

Integrar Scripts.

Integrar Plugins.

---

## Pode conter

- Toast System
- Theme Switcher
- Form Validation
- Infinite Scroll
- Lazy Loading

---

## Nunca deve conter

- Componentes duplicados
- Layouts duplicados
- Código específico do projeto

---

## Dependências

Foundation

Design System

Layout

Components

Scripts

Plugins

---

## Fluxo

Features utilizam Components para entregar funcionalidades.

---

## Boas práticas

Criar Features independentes.

---

## Anti-patterns

Duplicar Components.
