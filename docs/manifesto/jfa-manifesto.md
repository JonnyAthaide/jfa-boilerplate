# Manifesto

## Nossa missão

Criar aplicações Front-End organizadas, modulares e fáceis de manter.

---

## Princípios

- Simplicidade antes da complexidade.
- Arquitetura antes da implementação.
- Código explícito.
- Baixo acoplamento.
- Alta coesão.
- Documentação como parte do código.

---

## Filosofia

Toda camada possui apenas uma responsabilidade.

Toda Feature deve ser independente.

Toda dependência deve ser explícita.

A Foundation existe para servir às Features, nunca o contrário.
